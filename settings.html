Cách lấy API Key từ ERPNext

Hướng dẫn tạo API Token trong ERPNext

1
Mở User List — Vào /app/user trong ERPNext, chọn user bạn muốn tạo API key
2
Vào tab "Settings" — Kéo xuống phần API Access
3
Click "Generate Keys" — Hệ thống sẽ hiện popup API Secret. Copy ngay vì chỉ hiển thị 1 lần!
4
Copy API Key — Nằm ở field "API Key" trong cùng section
5
Dán vào đây — Paste API Key và API Secret vào form phía trên, click "Kiểm tra kết nối"
